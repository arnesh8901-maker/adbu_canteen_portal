### **📖 Project Overview**
This web-based portal was developed specifically for **Assam Don Bosco University** to digitize the traditional canteen menu and feedback mechanism. It empowers students to view live availability and share honest reviews, helping the management maintain high food standards.

---

### **🚀 Key Features**
- **🔐 Dual-Role Authentication:** Integrated separate portals for **Student Access** and **Canteen Administration**.
- **📜 Live Menu Tracking:** Real-time updates for food items, categories, and dynamic pricing.
- **⭐ Interactive Rating System:** Students can provide **1-5 Star ratings** along with detailed textual feedback.
- **🌙 Smart Dark Mode:** A professional UI toggle implemented with JavaScript for an enhanced night-time experience.
- **📱 Fully Responsive Design:** Optimized for seamless performance on **Mobile, Tablet, and Desktop** devices.

### **💻 Detailed Installation Guide**

Follow these steps to set up the project on your local machine:

#### **Step 1: Environment Setup**
* Download and install [XAMPP](https://www.apachefriends.org/).
* Open the **XAMPP Control Panel** and start the **Apache** and **MySQL** modules.

#### **Step 2: Project Deployment**
* Copy the entire project folder.
* Navigate to your XAMPP directory: `C:\xampp\htdocs\`.
* Paste the folder there (e.g., name it `adbu_canteen`).

#### **Step 3: Database Configuration**
* Open your browser and go to `http://localhost/phpmyadmin/`.
* Click on **New** and create a database named `canteen_db`.
* Click on the **Import** tab, select the `database.sql` file from the project's database folder, and click **Go**.

#### **Step 4: Launch the Application**
* In your browser, enter the following URL:
    `http://localhost/adbu_canteen/register.php`
